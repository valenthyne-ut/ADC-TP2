# ADC-TP2

Este trabalho faz parte do plano curricular do curso de Sistemas e Tecnologias de Informação da UAlg, do ano letivo de 2025/26, sendo o segundo trabalho prático a desenvolver em grupo da disciplina de Ambientes de Desenvolvimento Colaborativo.

Tem como objetivo aplicar os conhecimentos adquiridos em aula sobre o software de controlo de versões Git, e a utilização de algumas das funcionalidades do Github.
